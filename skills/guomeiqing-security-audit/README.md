# 🦞🔒 guomeiqing-security-audit

一句话让你的龙虾完成安全自检和加固。

## 安装

```bash
openclaw skills install npm:guomeiqing-security-audit
```

## 使用

对你的龙虾说以下任意一句话：

- 「安全检查」
- 「security check」
- 「帮我做个安全审计」
- 「安全扫描」

## 工作原理

这是一个 **prompt-only skill**（纯 SKILL.md），不包含任何脚本。所有操作通过 Agent 的内置工具完成：

- `read` — 读取配置文件
- `exec` — 检查文件权限、备份配置、重启 Gateway
- `edit` — 精确修改配置项

Agent 会自动：
1. **🔍 扫描** — 读取 `openclaw.json`，对 7 大安全维度逐项审计
2. **📊 报告** — 输出带评分的安全报告（0-100 分）
3. **💡 方案** — 生成三档加固方案（基础 / 标准 / 严格）
4. **🔧 执行** — 你选定方案后自动修改配置并验证

## 三档加固方案

| 方案 | 耗时 | 适合场景 |
|------|------|---------|
| 🟢 基础加固 | ~5 分钟 | 修复最危险的问题 |
| 🟡 标准加固 | ~10 分钟 | 推荐！兼顾安全与易用 |
| 🔴 严格加固 | ~15 分钟 | 最高安全，适合公网部署 |
