# 🦞 OpenClaw Threat Model Quick Reference

> 🚧 Work in Progress — This document will compile a comprehensive view of security threats facing OpenClaw.

## Planned Topics

- **Prompt Injection** — Manipulating Agent behavior through conversation content
- **Tool Misuse** — Exploiting Agent tool permissions to perform malicious operations
- **Data Exfiltration** — Stealing sensitive information through the Agent
- **Configuration Weakness** — Security risks in default configurations
- **Network Exposure** — Gateway and service port exposure
- **Supply Chain** — Trust chain issues with Skills / plugins
