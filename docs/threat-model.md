# 🦞 OpenClaw 威胁模型速查

> 🚧 编写中 — 本文档将整理 OpenClaw 面临的安全威胁全景。

## 计划内容

- **Prompt Injection** — 通过对话内容操纵 Agent 行为
- **Tool Misuse** — 利用 Agent 的工具权限执行恶意操作
- **Data Exfiltration** — 通过 Agent 窃取敏感信息
- **Configuration Weakness** — 默认配置的安全风险
- **Network Exposure** — Gateway 和服务端口暴露
- **Supply Chain** — Skill / 插件的信任链问题
